GyanApp — AI Study Helper 

GyanApp is an AI-powered platform that helps students in their studies.

    AI-Powered Branding – Instantly generate quizes, review answer sheets, leaderboard system for group study sessions.

    Authentication – Secure user login and registration powered by Firebase.

    Database – Firebase for storing users, brands, assets, and subscription data.

    Frontend – Responsive UI built with Flutter.

    Backend API – Node.js + Express handles AI generation, authentication, and data management.


    Versioning & Changelog – Semantic versioning with a clear project evolution record.

    Deployment Ready – Easily deploy frontend and backend with Firebasae integration.

Tech Stack

    Runtime: Node.js.

    Language: Flutter.

    Frontend: Flutter

    Database & Auth: Firebase (Database, Storage, Authentication).
    AI Service: Gemini API (Logo, Story, and Content Generation).

    HTTP Client: Axios/Fetch API.

    CI/CD: GitHub Actions (Automated Testing & Deployment).

    Hosting: Firebase

Quick Start
Prerequisites

    Node.js 16+

    Supabase project (for Authentication, Database, and Storage)

    Gemini API key (for AI-powered logo and content generation)
